# ApprenticeVR: Edición VRSrc

**ApprenticeVR: Edición VRSrc** es una aplicación moderna multiplataforma de escritorio construida con Electron, React y TypeScript para gestionar e instalar contenido en dispositivos Meta Quest. Se conecta a una biblioteca comunitaria de juegos, maneja descargas e instalaciones automáticamente y permite contribuir contenido de vuelta a la biblioteca.

> **Nota del fork:** Este fork incluye correcciones de errores, mejoras de rendimiento y nuevas funciones.

---

## Paso 1: Descargar el archivo correcto para tu sistema operativo

| Archivo | Plataforma | Notas |
|--------|-----------|------|
| `apprenticevr-x.x.x-x64.dmg` | macOS | Funciona en Intel y Apple Silicon (Rosetta 2) |
| `apprenticevr-x.x.x-setup-x64.exe` | Windows | Recomendado |
| `apprenticevr-x.x.x-portable-x64.exe` | Windows | No requiere instalación |
| `apprenticevr-x.x.x-x86_64.AppImage` | Linux x64 | Compatible con la mayoría de distros |

> Descarga siempre la versión más reciente.

---

## Paso 2: Obtener credenciales del servidor

Necesitas:

- `baseUri`
- `password` (codificada en base64)

Fuentes:

- Telegram: https://t.me/the_vrSrc
- Vista web: https://t.me/s/the_vrSrc

---

## Paso 3: Introducir credenciales

### Opción A (recomendada)

1. Abrir Settings
2. Click en **Set Public Server JSON**
3. Pegar JSON
4. Guardar

---

## Paso 4: Conectar Quest y usar sideload

1. Conectar por USB
2. Permitir depuración
3. Descargar juegos
